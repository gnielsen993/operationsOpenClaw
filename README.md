# Operations OpenClaw

Public-safe operating reference for user's OpenClaw/Jarvis setup.

This repo is designed to answer: **"What can my assistant do, how do I invoke workflows, and what guardrails keep it safe?"**

## What belongs here
- Public-safe capability summaries
- Workflow trigger references
- Repo and release operating rules
- Automation schedules and expected outputs
- Sanitized setup notes for tools like Obsidian

## What does not belong here
- API keys, tokens, private keys, credentials, account IDs
- Raw chat transcripts or private memory dumps
- Personal/private context not needed for public reference
- Sensitive client/project details
