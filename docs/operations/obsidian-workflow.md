# Obsidian Workflow

Obsidian is the clean human-facing operating notebook.

## Role split
- OpenClaw memory: private raw machine continuity
- Obsidian: readable dashboards, daily wins, decisions, project state
- operationsOpenClaw GitHub repo: public-safe quick reference

## What to track in Obsidian
- Active priorities
- Daily wins
- Project dashboards
- Decisions
- Automation status
- Release logs

## Keep out of Obsidian
- Secrets
- Raw transcripts
- Sensitive context not needed for operations
